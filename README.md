## 🚀 Projects

### 💰 [Budget Tracker](https://github.com/Cameron-Guinnee/Budget-Tracker)
A personal finance dashboard that connects directly to Google Sheets for live data, with interactive visualizations across income, expenses, categories, and trends.

**[→ Live App](https://budget-tracker-hqzvmiazybvhhkjkv4pih8.streamlit.app/)**

- Built with **Streamlit**, **Pandas**, **Plotly**, and **gspread**
- Full read/write integration with the Google Sheets API via a service account
- Features: spending breakdowns, monthly trends, expense heatmaps, word clouds, and in-app transaction logging
- Containerized with **Docker** for portable deployment
